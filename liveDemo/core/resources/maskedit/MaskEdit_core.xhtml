Demonstrated Features

  • Customized mask form and separators location - You can configure the original mask form and the input separators location and note the symbols that you can input in any cases.
  • Displaying blank - It is possible to make blank visible by specifying blankVisible attribute.
  • Features of the JSF InputText component - The MaskEdit component supports all of the features of the JSF InputText component standard that banes to input the symbols into the blankText that are allowed in the input mask and has features like rollover and focused styles.
