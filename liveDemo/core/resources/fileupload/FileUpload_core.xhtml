Demonstrated Features

  • Client side interaction - Slider provides client side API to manage component's value.
  • Different orientations of Slider - Supports horizontal and vertical orientations.
  • Provides full managing of ticks - Ticks can be located on left, right or both sides of slider. Slider's value might be set at specific range. Also provided settings of distance between of big and small ticks. There is oportunity to not displaying ticks.
  • Customizable styles - Slider bar and drag handler can be replaced by another images. Width and height can be customized for each slider. Separate styles for component and for label that shows current control's value.
