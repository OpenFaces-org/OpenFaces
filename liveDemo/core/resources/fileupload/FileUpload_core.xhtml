Uploaded files:

Demonstrated Features

  • Provides support for uploading several files - add several files with 'Add' button and then click 'Upload' button.
  • Customized styles - you can style almost every part of FileUpload component: each button in header, FileUpload item which represents filename, its progress, status, remove/stop/cancel button and customize them separately.
  • Ability to limit file size - try to upload file bigger than 200MB.
  • Provides an ability to stop uploading of each file - while file is uploading, click on 'Stop' button. If it's not yet uploaded on server, file uploading will be stopped.
  • Support for addding several files at the time - by standard file dialog or drag and drop. Just drag file(s) to the browser and drop them at fileUpload drop target.
