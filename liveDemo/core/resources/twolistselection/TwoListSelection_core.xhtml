Click the Update button above the Tasks list to apply changes made in the upper TwoListSelection component.

Customization Options

Selecting items in this TwoListSelection component affects the appearance and capabilities of the upper TwoListSelection component.

Demonstrated Features

  • Items ordering and sorting - You can reorder selected items by using the Up and Down buttons on the right and sort them by clicking the header of the right-hand list box. Ordering and sorting can be disabled by selecting a corresponding option in the bottom TwoListSelection component.
  • Items selection - You can select and remove individual, multiple or all items. Use the bottom TwoListSelection component to enable/disable selection and removal of all items.
  • Changing labels of buttons and captions - All text labels of the buttons and captions can be changed by selecting a corresponding option in the bottom TwoListSelection component.
  • Styles customization - You can change the styles of the upper TwoListSelection component by selecting a corresponding option in the bottom TwoListComponent. Note that the Safari browser does not support customization of some styles for list boxes.
