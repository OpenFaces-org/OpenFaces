Demonstrated Features

  • Sorting - Click any column header in the DataTable component to sort the column data in ascending or descending order.
  • Column menu - You can change sorting order and show/hide columns in the column menu invokable from column's header.
  • Single and Multiple Row Selection - Click a row to select it. Or, select a number of rows by holding down the "Ctrl" key (or under Mac OS) while clicking the required rows.
  • Keyboard Navigation - Use the "Up Arrow" and "Down Arrow" keys to move selection one row up and down. Or, navigate between rows with the "Shift" pressed to select adjacent rows. Use the "Home" and "End" keys to move selection to the first and last rows.
