Demonstrated Features

  • Static tree structure - All the nodes are specified right on the JSP page.
  • Preloading nodes - All the nodes are preloaded, which means that you can expand them without sending any requests to the server.
  • Hierarchical selection - You can make a bulk selection by checking or unchecking parent check-boxes, and customize selection for each node individually.
  • Customization of the default expansion state - Only the nodes at the two top levels are expanded by default when the TreeTable component is loaded.
  • Customization of the expansion toggle images - In this demo, the standard +/- expansion toggle images are replaced by custom ones.
