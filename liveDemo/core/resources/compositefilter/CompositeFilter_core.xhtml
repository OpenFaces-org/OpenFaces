Demonstrated Features

  • Filter composition - Click on '+' button to add new filter, select property, operation and input parameters. Click on '-' button to remove filter row.
  • Attachement to Data Table - Table data is automatically filtered according to conditions configured by Composite Filter.
  • Multiply data types - Composite Filter provides different sets of operations and different appearance of parameter editor according to the type of the property.
  • Auto columns detection - Composite Filter automatically fetches all columns from the attached Data Table and defines their types.
