Draggable Resizable Modal Shadeable Customizable

Demonstrated Features

  • Drag-and-drop - Click on the Drag-and-droppable icon and you can see the draggable window, that can be dragged around on the page
  • Resizing - Click on the Resizable icon and you can see the resizable window
  • Modality - Click on the Modal icon and you can see the modal window
  • Customization of the area that surrounding the modal window - Click on the Shadeable icon and you can see the modal window with customized visible part of the page which is not covered by a Window component
  • Support for custom caption buttons - Click on the Customizable icon and you can see the window with customized caption buttons
